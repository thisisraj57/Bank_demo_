**Documentation for Account class:**

**Class:** Account

**Description:**
Represents an account in the system.

**Attributes:**

- **id:** Unique identifier for the account.
- **name:** Name of the account holder.
- **email:** Email address of the account holder.
- **phoneNumber:** Phone number of the account holder.
- **status:** Status of the account (e.g., approved, rejected, pending).

**Getters and Setters:**
Standard getters and setters are provided for all attributes.

**Example Usage:**

```java
// Create a new account
Account account = new Account();
account.setName("John Doe");
account.setEmail("johndoe@example.com");
account.setPhoneNumber("123-456-7890");
account.setStatus("approved");

// Get the account's name
String name = account.getName();

// Set the account's email address
account.setEmail("new_email@example.com");
```

**Additional Notes:**

The `@Id` and `@GeneratedValue` annotations are used for defining the primary key and its generation strategy, respectively. The `GenerationType.IDENTITY` indicates that the ID should be auto-generated by the database.
