**Function:** print

**Description:**
The `print` function is used to display output to the console.

**Syntax:**

```
print(*objects, sep=' ', end='\n', file=None, flush=False)
```

**Parameters:**

* **objects:** The objects to be printed. These can be any type of object, including strings, numbers, lists, and dictionaries.
* **sep:** (optional) The separator to be used between the objects. The default separator is a single space.
* **end:** (optional) The character to be printed at the end of the output. The default end character is a newline.
* **file:** (optional) The file to which the output should be written. The default file is the console.
* **flush:** (optional) A Boolean value indicating whether the output should be flushed immediately. The default value is `False`.

**Example:**

```python
print("HELLO WORLD")
```

**Output:**

```
HELLO WORLD
```
